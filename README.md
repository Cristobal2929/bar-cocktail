# Bar El Refugio

Esta es una página estática de una sola página (`index.html`) pensada para el bar **El Refugio**.  
Incluye:

- Tipografía personalizada (Playfair Display + Montserrat) cargada desde Google Fonts.  
- Hero visual con fotografía de ambiente y mensaje propio del bar.  
- Menú de cócteles con tarjetas visuales.  
- Enfoque de teclado visible y soporte para `prefers-reduced-motion`.  
- Responsividad móvil y escritorio sin frameworks ni build tools.  

Para usarla, simplemente abre `index.html` en cualquier navegador. No se requieren dependencias externas más allá de los recursos de Google Fonts y las imágenes alojadas en Unsplash.
